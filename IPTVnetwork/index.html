So You Want To Make Your Own IPTV Network

I cant blame you its a good idea plus its ton of fun. In the following guide I'll give you all the information you need to know plus some tips to make your network as successful as possible

Prerequisites

There are a couple things you'll need to intall or have pre installed if you want to run your own IPTV network. I'll go ahead and list everything you need and where you can install or purchase the things you need

A Computer For Running Your Network

You might already have a computer laying around that you can run this on but if you dont thats okay. Ideally you would use a desktop computer for this but there is nothing keeping you from using a laptop.

If you dont have a spare desktop computer or laptop laying around thats fine. Luckily its really easy to find one that can run your network for under 100$

You can typically find a computer for running your server from places like

  • Ebay - Ebay can be good and bad at the same time on one side you can get scammed or can pay a bit too much for an old office PC but on the other side you can find a pc thats good enough for your IPTV network for a good price. Just make sure you do your research on the seller and what they're selling

  • Thrift Stores - This is where I got my IPTV PC from. Just check all the ones near you for a desktop pc. if you get lucky you might find one for a good price (for example I got mine for 25$) The main thing you want to look out for when shopping for an IPTV PC at a thrift store is buying a PC that doesnt work. Before you buy your PC from a thrift store make sure you plug it in for a second to make sure it cuts on. If you're really really anxious that you're going to get a computer that doesnt work then you can plug it into a monitor if the thrift store has it

    IMPORTANT INFORMATION TO NOTE

    if you buy a computer from a thrift store sometimes there will still be data on it. When you get this computer DO NOT look through that persons personal documents. My computer had someones stuff on it when I got it. I personally looked for their email so I could email them to see if they wanted any of their data back. You dont have to do this if you dont want to. Nor do you have to go through and delete all their stuff. We're going to be wiping their data from their computer regardless

  • Pawn shops - They're pretty similar to thrift stores when it comes to getting your PC. You just have to watch out for places that might try to overcharge you. Now this isn't every pawn shop but some of the people that work at these places are non technical and they might try to overcharge you for a bad PC just because they think its a good "gaming PC" do your research on the specs on whatever you're going to buy. Dont spend 200$ on an office PC made before NVIDIAs 2000s series.

Docker

Docker is a program that lets people create and ship "containers" that contains the code for a program and everything else the program needs. We're going to use this to install the software we need for our IPTV network

You can install docker here from the "Next steps > Install docker desktop" then you choose your Operating System

You should create an account with docker or sign in if you already have an account if you need a guide on how to install docker desktop then just google it. googling is an essential skill for this.

Choosing an operating system

Depending on what your original operating system is and how powerful your computer is you might want to change your operating system. I've set up everything twice and I'll list the pros and the cons for each programming sytem (that matters. MacOS doesnt matter.)

after reading over the pros get a flash drive and flash your choice of operating system onto your computer

Linux

Pros
  • Practically runs on a toaster
  • You get more control with your operating system
  • You can install other linux exclusive apps
Cons
  • Is more complex (kind of)
  • practically unusable if you dont have access to google

Windows 11/10

Pros
  • more simple and user friendly
Cons
  • Uses more system resources by default meaning your TV network could run slower
  • There are tons of different versions of windows with different features missing on each one

Setting up ErsatzTV

once you have ersatzTV installed and running you can navigate to it by going to 127.0.0.1:8409

First you want to add the content you're going to have on your network into one folder. Theres a specific way you have to set up your folder or it will not work.

(note from me > I cant take a photo of my own setup right now but take this as text)

the folder of your content should look something like this

Home folder
- Tv show 1
-- Season 1
--- s1e1
--- s1e2
-- Season 2
-- metadata.nfo
- TV show 2
-- Season 1
-- Season 2

PLEASE NOTE THAT THE SEASONS MUST BE CALLED

"Season X" (x is the number of the Season)

AND THE NAME OF THE TV SHOW FOLDER MUST MATCH WITH THE NAME IN metadata.nfo

SHOWS MUST ALSO BE NAMED FOLLOWING sXeY OR ELSE THEY WILL NOT WORK



You can find the docs for NFO files here

Then you can add the folder to Ersatz by following this guide

once you add the path with all your media to the network you must now add your media to a collection
You can find out how to add your media to a collection here
